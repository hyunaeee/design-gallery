유명 디자인을 코드로 배우는 갤러리

웹·앱의 검증된 디자인 패턴을 라이브 데모와 복사 가능한 코드로. 카드를 열면 미리보기·소스·피그마 토큰·네이티브 코드가 있습니다.

새 탭 ↗
🎨 피그마 적용 — 토큰 JSON: 피그마에서 Tokens Studio 플러그인 열기 → Tools → Load from JSON에 붙여넣으면 색상·radius·타이포가 변수/스타일로 자동 생성됩니다. 화면 자체는 html.to.design 플러그인에 이 데모 URL을 붙여넣으면 편집 가능한 프레임으로 들어옵니다.
📱 앱 적용 — RN은 Expo Snack(snack.expo.dev), Flutter는 DartPad(dartpad.dev)에 붙여넣으면 즉시 실행, SwiftUI는 Xcode 새 파일에 붙여넣으면 #Preview가 바로 뜹니다. 전부 외부 패키지 0개. 토큰 JSON은 Style Dictionary로 네이티브 상수 변환도 가능합니다.
